Étape 1 / 4

👋 Bienvenue

Cet outil te permet de suivre tes heures supplémentaires mois par mois, avec report automatique et estimation €.

La configuration prend moins d'une minute.

  1. Clique sur un jour pour saisir des heures
  2. La clôture mensuelle est le dernier dimanche par défaut
  3. Les reliquats sont reportés automatiquement
  4. La zone € est protégée par un code

📆 Clôture mensuelle

La clôture définit la fin de ta période de paie.
Par défaut : dernier dimanche du mois.

📅 Clôtures des 12 mois

Choisis un dimanche par mois. Laisse vide = dernier dimanche automatique.

💶 Taux horaire

Ces valeurs pré-remplissent la zone €. Modifiables à tout moment.

Laisser vide = valeurs par défaut.

🔐 Code de sécurité

La zone € est protégée par un code. Le code par défaut est 1234 — change-le pour sécuriser tes données.
